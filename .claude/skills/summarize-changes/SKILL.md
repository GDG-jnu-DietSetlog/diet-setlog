## 현재 미커밋 변경사항

!`git status --short`

!`git diff HEAD`

## 지시

변경사항을 2~3개 불릿으로 요약하라. 그다음 위험요소를 나열하라:

- 누락된 에러 처리 / 입력 검증(zod)
- 하드코딩된 값·시크릿·매직 넘버
- 갱신이 필요한 테스트
- 앱↔서버 API 계약(요청/응답)의 호환성 깨짐
- 트래픽 스파이크 가정 위반(N+1 쿼리, 무캐싱 핫경로, 상태 보유 등)

변경이 없으면 "변경 없음"이라고 답하라.
